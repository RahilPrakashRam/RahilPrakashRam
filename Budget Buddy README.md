# BudgetBuddy

BudgetBuddy is a Flutter app designed to help users track their expenses and visualize their spending habits using interactive charts. Whether you want to monitor your daily, weekly, or yearly expenditures, BudgetBuddy provides intuitive tools to manage your finances effectively.

## Features

- **Expense Tracking**: Keep a detailed record of your expenses with customizable categories.
- **Chart Visualization**: View your spending patterns through visually appealing charts for daily, weekly, and yearly periods.
- **Category Management**: Create and manage expense categories, assigning different colors for easy identification.
- **User-Friendly Interface**: Intuitive design and smooth navigation make managing your budget effortless.
- **Data Backup**: Securely store your financial data to prevent loss and ensure continuity across devices.
- **Budget Planning**: Set financial goals and track your progress over time.

## Getting Started

### Prerequisites

- Flutter SDK installed on your development machine. [Install Flutter](https://flutter.dev/docs/get-started/install)
- Android Studio / IntelliJ or Visual Studio Code with Flutter plugin for development.
- A device or emulator to run the app.

### Installation

1. Clone the repository:

    ```
    git clone https://github.com/your-username/budgetbuddy.git
    ```

2. Navigate to the project directory:

    ```
    cd budgetbuddy
    ```

3. Install dependencies:

    ```
    flutter pub get
    ```

4. Run the app:

    ```
    flutter run
    ```

## Usage

1. **Adding Expenses**:
    - Tap on the "+" button to add a new expense.
    - Enter the amount, select a category, and add a description if necessary.
    - Save the expense to record it in the app.

2. **Viewing Charts**:
    - Navigate to the "Charts" section to visualize your expenses.
    - Choose between daily, weekly, or yearly charts to see your spending trends.
    - Use gestures to interact with the charts for a better understanding of your finances.

3. **Managing Categories**:
    - Access the "Categories" section to create, edit, or delete expense categories.
    - Assign different colors to categories for easier identification.

4. **Setting Goals**:
    - Define your budget goals in the "Budget Planning" section.
    - Track your progress over time to stay on target and achieve financial stability.
